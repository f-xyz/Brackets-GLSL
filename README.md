Brackets-GLSL
=============

[Brackets](http://brackets.io/)でHTMLに含まれる頂点シェーダ(`<script type="x-shader/x-vertex">`)、フラグメントシェーダ(`<script type="x-shader/x-fragment">`)を色分け表示します。

### インストール方法

拡張機能マネージャのURLからインストールから

 https://github.com/sapphire-al2o3/Brackets-GLSL

を入力してインストールする。
